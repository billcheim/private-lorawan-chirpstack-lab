# Contributing

Contributions are welcome.

## How to contribute

1. Fork the repository
2. Create a branch
3. Keep changes focused
4. Commit clearly
5. Open a pull request

## Helpful areas

- ChirpStack deployment notes
- Gateway configuration
- Sensor onboarding workflows
- MQTT integrations
- Dashboards
- Automation scripts

## Please include

- OS version
- ChirpStack version
- Gateway model
- Logs if relevant
