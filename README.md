# Private LoRaWAN Sensing Network with ChirpStack

Documentation and deployment-oriented lab guide for building a private LoRaWAN sensing network using:

- ChirpStack
- LoRaWAN gateway
- sensor end nodes
- MQTT integration
- private network server

---

## Overview

This repository documents a practical lab workflow for building a private LoRaWAN network for sensing applications.

It is intended for:

- IoT labs
- LPWAN experimentation
- smart sensing demos
- telemetry systems
- research deployments

---

## Features

- ChirpStack installation notes
- gateway setup guidance
- sensor node onboarding
- MQTT integration
- troubleshooting guide
- private sensing network architecture
